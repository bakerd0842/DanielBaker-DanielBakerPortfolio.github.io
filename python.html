Python Unit 1

Variables Lab

Description:

The Variables Lab is actually comprised of two segments, “variables.py” and “math.py”, the former dealing with the user’s name, and the latter primarily focusing on the dynamics of math in Python, and familiarising the user with said dynamics, but in each, the program integrates an input and prints the output of its respective code. Within the lab, the rudiments of Python, such as printing and the creation and manipulation of variables are introduced.

Concepts Learned

  • Receiving input originating from the user
  • The logistics of printing within Python
  • The logistics of variables
  • Basic Mathematical concepts like addition, subtraction,multiplication, and addition
  • Modulus (I was not familiar with the concept prior)

    • Python Unit 2

    Collide Lab

    Description:

    In this assignment, the character defined in the previous labs with the same characteristics and methods is programmed in order to interact properly with another impassible object, that being a wall. Subsequently, the use of “if/else” statements is necessitated, and so they are introduced in this lab

    Concepts Learned

    • If/else statements
    • Incorporation of another object into pygame
    • Interaction (including collision) of multiple objects, using ChangedRecs
    • Use of multiple class files

      • Python Unit 3

      Timer Lab

      Description:

      In this particular lab, a program should be written that counts down from a numerical value that the user defines to zero. The numerical value that user defines will be done so through asking the user, and the pygame timer will be utilised to pause the loop for 1 second. The continual decrease from the set value to zero will be displayed in seconds first and then in seconds.

      Concepts Learned

      • Use of the Pygame Timer
      • Use of loops to repeat code a definitive number of times
      • Representing time in Python

        • Python Unit 4

        Costume Change Lab

        Description:

        The objective of this lab is to devise a program with the capacity to change the subject’s image according to the direction in which it is facing. Since the subject is to face the direction in which it is moving, the image will correspond to the direction in which it is moving, as well. Each particular costume will be invoked through the use of a list of costume and a particular index that corresponds to each individual element in the entire set.

        Concepts Learned

        • The loading of multiple image files to a particular object (permitted through a list)
        • The basic concept of creating lists
        • The use of indices to invoke their respective elements

          • Python Final Project

          Costume Catastrophe

          Description:

          Within this particular game, one is a assigned a red shirt to control with the “a” and “d” keys, to move left and right, respectively. With this capacity to move, the player must pursue green shirts and avoid blue shirts, which will increase and decrease the player’s score, respectively. The player will cycle through multiple levels, and with each successive level, the pace would increase.

          Concepts Learned

          • The displaying of multiple variables onto the screen
          • The interaction of numerous variables
