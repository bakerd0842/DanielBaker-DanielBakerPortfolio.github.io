    Great Escape

    Description:

    In this Unit, there were the labs Escape, Flowers, LED, Water Hazard-1, and Box. In this particular unit, the concepts learned include how to create a method and the differences between coding within the main method versus the side methods. This difference manifests in that in the main method, one must place (enter Jeroo name).(x function), while in a side method, one can simply format it as (x function). In addition, we learned to apply a particular method multiple times

    Concepts Learned

    • How to create a method
    • The differences between coding in the main method and side methods
    • Applying the same method multiple times for one particular Jeroo

        • Jeroo Unit 4

        Alley Run

        Recursive Box

        Description:

        In this Unit, there were the labs Chaos, RecursiveBox, Spiral, and AlleyRun. In this particular unit, the concepts learned include how to use an additional conditional, the "If/else" statement, as well as Recursion. Recursion is a very helpful computing trick in which, upon being activated and having the rest of its content fulfilled through another method, a particular method will invoke the original method that triggered it again. This perpetuates the operation of the code, although particular factors can still be integrated to be fulfilled at a particular point to prevent further Recursion

        Concepts Learned

        • How to establish a Recursive Statement
        • How to check for multiple variables concurrently,and be able to act accordingly
        • Using the "If/else" statements
